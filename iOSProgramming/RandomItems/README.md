12. 为初始化方法总结若干简单的规则：  

    - 类会继承父类所有的初始化方法，也可以为类加入任意数量的初始化方法  

    - 每个类都要选定一个指定初始化方法  

    - 在执行其它初始化工作之前，必须先用指定初始化方法调用父类的指定初始化方法（直接或者间接）  

    - 其他初始化方法要调用指定初始化方法（直接或者间接）  

    - 如果某个累所声明的指定初始化方法和其父类不同，就必须覆盖父类的指定初始化方法，并调用
    新的指定初始化方法（直接或间接）
